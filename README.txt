To Successfully use the BookTracker.java application follow the following steps:

1. Make sure the direction contains the following:
    The folder "lib" containing the jdbc driver jar file.
    The following files:
        Booktracker.db (if the file is not present, follow step 1.5 otherwise skip)
        Booktracker.java
        main.java

1.5. if Booktracker.db is not present in the directory, make sure to do the following:
    Make sure the files "User.csv" and "ReadingHabit.csv" are present in the directory. (if not follow step 1.75)
    Run BookTracker.java and watch booktracker.db magically appear.

1.75. if User.csv and ReadingHabit.csv are not present in the directory, make sure to do the following:
    Open the provided xlsx file. 
    choose the Readinghabit table in the drop down menu
    choose "save as" and select "csv" and name the file "ReadingHabit"
    do the same for the 2nd table present in the xlsx file and name it "User"

2. Run main.java and follow the steps given in the terminal. Enjoy!
